事件

:::demo
```html
<template>
  <nb-verify
    text="请滑动王俊至右边"
    :isPassing.sync="isPassing"
    @passcallback="passcallback"
    successText="验证通过"
    ref="dragVerify"
  />
  <br/>
  <nb-button @click="reset" text="重置"/>
</template>
<script>
  export default {
    data(){
      return{
        isPassing:false,
      }
    },
    methods:{
      reset(){
        this.isPassing = false;
        this.$refs.dragVerify.reset();
      },
      passcallback(){
        console.log("验证成功")
      }
    }
  }
</script>
```
:::

### Attributes
| 参数      | 说明          | 类型      | 可选值                           | 默认值  |
|---------- |-------------- |---------- |--------------------------------  |-------- |
| isPassing     |      是否通过     | boolean | — | false |
| width | - | string/number |  | |
| height | - | string/number | — | — |
| text | - | string | — |  |
| successText | - | string | — | - |
| background | - | string | — | — |
| progressBarBg | 滑动时背景 | string | - | - |
| completedBg | 成功背景 | string | true/false | flase |
| circle | - | boolean | true/false | flase |
| radius | - | string | - | 4px |
| handlerIcon | 滑块上icon | string | - | - |
| successIcon | - | string | - | - |
| handlerBg | 滑块颜色 | string | - | - |
| textSize | - | string | - | - |
| textColor | - | string | - | - |
